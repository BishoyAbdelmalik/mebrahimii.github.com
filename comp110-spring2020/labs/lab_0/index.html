Step-by-Step Instructions

Step 1: Edit the lab0questions.txt File

Using a text editor of your choice, open the lab0questions.txt file. The file contains a series of questions for you to answer. The answers should be placed directly in the file itself. Any answers are valid for this lab, though you must provide an answer for full credit. Make sure you save your answers before you exit.

Step 2: Edit the lab0_quiz_questions.txt File

Using a text editor of your choice, open the lab0_quiz_questions.txt file. You need to put answers into this file, based on the question set here. The answers should be placed directly in the file itself. Any answers are valid for this lab, though you must provide an answer for full credit. Make sure you save your answers before you exit.

Step 3: Turn in Your Answers Using Canvas

Log into Canvas, and go to the COMP 110L class. Click “Assignments” on the left pane, then click “Lab 0”. From here, you need to upload the following two files:

  • lab0questions.txt
  • lab0_quiz_questions.txt

Be sure that the filenames are correct, and that they are both plain text files.

You can turn in the assignment multiple times, but only the last version you submitted will be graded.
